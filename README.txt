Please folow step by step:

I: Backend Quick Install

  1. Open your shell in this folder
     $cd backend

  2. Install modules
     $npm install

  3. Run server
      $npm start

  4. Run unit test
     $npm test


II: Front-end Quick Install

  1. Change direction to frontend folder on Shell
     $cd ../frontend 
    ($cd frontend if current direction is this folder)

  2. Install modules
     $npm install

  3. Run server
     $npm start

III: How to use:

  1. Open your browser and enter address: 
      localhost:3000

  2. Main screen is login screen
      - Admin account: 
          username: admin
          password: admin
      - User account:
          username: ntnbker
          password: ntnbker
      - Register:
          click button register and follow to register new user (role default is 'user')

  3. When login successful, main screen change to Dashboard
      * Main feature:
        - Find: Find user that name match your input value
        -  My Profile: View and Updates your profile
        - Logout: ......

  4. Click a User or My Profile, this screen will change to detail profile:
      * Save button:
          - Save your new profile
          -  If your role is user: your can only change your profile
          -  If your role is admin: your can change anyprofile

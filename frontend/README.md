# README #
ReactJS application starterkit( React, Redux, Fetch ...)
 
### Quick start ### 

Install node version lastest:

Use NVM:

nvm install node
```

Install node_modules from package.json:

```
#!shell

sudo npm install
```

### Build, compile and deploy ###

To compile for development, type:

```
#!bash

npm start

```
To compile for deployment, type:

```
#!bash

npm build
```
